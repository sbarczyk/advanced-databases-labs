# Bazy Danych II — Laboratoria

Zestaw zadań i implementacji zrealizowanych w ramach przedmiotu **Bazy Danych II** na **Akademii Górniczo-Hutniczej w Krakowie**, Wydziale Informatyki.

Repozytorium zawiera ćwiczenia praktyczne obejmujące różne technologie baz danych:
- **Oracle SQL** – projektowanie relacyjnych baz danych i tworzenie zaawansowanych zapytań,  
- **MongoDB** – modelowanie danych w bazach dokumentowych i operacje agregujące,  
- **Entity Framework** – mapowanie obiektowo-relacyjne i komunikacja z aplikacjami .NET.  

Każdy folder odpowiada zestawowi laboratoriów, których celem było pogłębienie wiedzy z zakresu modelowania, zapytań i zarządzania danymi w różnych środowiskach.
